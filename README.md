# EL2 Mods

Collection of small, focused mods for **Endless Legend 2**.

Each mod solves a specific problem or adds a narrowly scoped quality-of-life feature, with an emphasis on stability, transparency, and respect for the game’s core systems.

Mods are built using **BepInEx** and **Harmony** and target single-player unless explicitly stated otherwise.

---

## 📦 Available Mods

- **Quest Recovery**  
  Recover stuck major faction quests  
  → https://github.com/AdBergman/EL2Mods/tree/main/EL2.QuestRecovery

- **WorldGen** 
  Improved terrain and elevation tuning

- **BulkTrade**
  Quality-of-life improvements for trading

- **SyntheticHarmony**
  AI and approval balancing improvements

---

## 📥 Installation (All Mods)

### 1. Install BepInEx
1. Download **BepInEx 5.x (Windows x64)**  
   https://github.com/BepInEx/BepInEx/releases  
2. Extract into your *Endless Legend 2* install folder  
3. Launch the game once

Typical path:
```
C:\Program Files (x86)\Steam\steamapps\common\ENDLESS Legend 2\
```

---

### 2. Install a Mod
1. Download the mod from the **Releases** page  


2. Extract into the same folder as `ENDLESS Legend 2.exe`  
3. Ensure the DLL ends up in:
```
BepInEx/plugins/
```

## 📦 Modpack

### EL2 Essentials Pack

A curated bundle of mods designed to improve the overall gameplay experience without changing the core identity of the game.

**Direct download:**  
https://github.com/AdBergman/EL2Mods/releases/tag/essentials-pack-v1.0.0

Includes:

- **WorldGen**  
  Makes world generation more varied, slightly increases map size, and ensures water remains after the final monsoon.  
  *(Works best on larger maps)*

- **BulkTrade**  
  Enables bulk trading of resources:  
  - Shift + Click → buy/sell 10x  
  - Ctrl  + Click → buy/sell 50x  

- **SyntheticHarmony**  
  Improves AI competitiveness by stabilizing approval:  
  - +25 approval at game start  
  - +5 approval every 10 turns  
  This helps compensate for the AI’s poor approval management.

Designed to work together as a cohesive gameplay improvement set.
