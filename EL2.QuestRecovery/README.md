# Quest Recovery (Endless Legend 2)

Quest Recovery is a small, single-player mod for Endless Legend 2 that allows you to manually progress stuck quest objectives.

It is intended as a recovery tool for broken or stalled quests, not as a cheat or automation mod.

## Scope & Limitations

- Single-player only
- Manual, one-step progression
- No automation or bulk completion
- No save-file editing
- Not supported in multiplayer

## Installation

### 1. Install BepInEx
1. Download BepInEx 5.x for Windows (x64)
2. Extract into your Endless Legend 2 install directory
3. Run the game once to generate folders

### 2. Install Quest Recovery
1. Download `QuestRecovery.dll`
2. Copy it to:
   `Endless Legend 2/BepInEx/plugins/`
3. Launch the game

## Usage

- Open the Quest window in-game
- A small "Quest Recovery" panel appears
- Click "Skip current quest step" to manually advance the current quest objective
- The action is one-shot and requires confirmation

## Uninstall

Remove `QuestRecovery.dll` from the `BepInEx/plugins` folder.

## Notes

Always back up your saves before using mods.
